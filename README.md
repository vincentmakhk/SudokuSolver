# SudokuSolver

Console C# program to solve Sudoku

## Status

It's not completed yet: it can only solve simple ones. 

## Usage

Input: a 9x9 text matrix:

```
  5  2   
 2 734   
 6 8    9
 9 1287  
  7  6  2
 84  31  
 7 35  8 
   4  2  
5     61 
```

Output: a filled 9x9 text matrix:

```
415692378
928734561
763815429
396128745
157946832
284573196
672351984
831469257
549287613
```
